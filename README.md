### Prerequisites

- node 12.16.3 _(required to use the npm commands)_
- npm 6.14.4
- MongoDB 4.4

### Getting Started

1.  Go to /frontend file, `cd frontend` and run README.md
2.  Go to /backend file, `cd backend` and run README.md
